	https://drive.google.com/drive/folders/0ByJcjwXw8XanQ3hRNk1uZW42M2M?usp=sharing
	it's my public google drive folder for this project
	
Exporation:

	exploration directory holds classes and methods which help to analyse data structure and build models.
	
App:

	launch run.py file. It has simple interface:
		- add model -> add g1.csv, g2.csv, g3.csv, g4.csv, g5.csv, g6.csv files from /models directory
		- learn -> you will see log in app frame with metrics and errors
		- predict -> add /t1 folder from /data
		- select file and press image button -> you will see image in app frame
